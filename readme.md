How to create a virtualenv
```
mkdir myproject
cd myproject
python3 -m venv venv
```

Activate the environment (venv)
```
. venv/bin/activate
```

Install Flask
```
pip install Flask
```

To install dependencies: 
pip install -r ../requirements.txt

to freeze:
pip freeze > requirements.txt

to Start server:
cd .. 
python web_server.py
